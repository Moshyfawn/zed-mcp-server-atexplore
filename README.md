# ATExplore MCP Server Extension for Zed

Zed extension that connects to the [ATExplore MCP server](https://mcp.atexplore.social) for exploring the AT Protocol ecosystem.

## Installation

Navigate to **Zed** > **Extensions** and search for "ATExplore".

Requires Node.js in PATH.

## Configuration (Optional)

```json
{
  "context_servers": {
    "mcp-server-atexplore": {
      "settings": {
        "server_url": "https://mcp.atexplore.social/mcp"
      }
    }
  }
}
```
